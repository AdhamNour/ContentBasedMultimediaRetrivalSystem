after extracting the files, do the following

for backend (ensure that you install python):
1. in terminal, go into backend folder by typing "cd ./multimedia_backend"
2. install all requirements by typing "pip install -r requirements.txt"
3. run the backend by typing "python ./app.py"

for frontend (ensure that you install flutter):
1. in terminal, go into frontend folder by typing "cd ./multimedia_frontend"
2. install all requirements by typing "flutter pub get"
3. either connect the mobile phone to your computer, or start an android emulator
4. run the frontend by typing "flutter run"
